## ワークフロー支援

### cc-advisor（ベストプラクティスアドバイザー）
- ユーザーのメッセージを受け取ったら、タスクに着手する前に `cc-advisor` サブエージェントでプロンプトを分析する
- サブエージェントには、ユーザーのプロンプト全文と現在のセッション状況（タスクの種類、セッションの長さ等）を渡す
- 返ってきたアドバイスがあればユーザーに簡潔に共有してからタスクに進む
- 「特になし」の場合はアドバイス表示をスキップしてそのままタスクに進む

## 言語・説明ルール
- ツール呼び出し（Bash, Edit, Write等）の前に、何をするか・なぜするかを日本語で簡潔に説明すること

## 外部ツール連携ルール

### GitHub
- GitHub操作は必ず `gh` CLI を使うこと
- GitHub MCP serverは使用禁止
- コマンドの詳細は `/github` を参照
